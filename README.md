# Dashboard d'Analyse du Marché des Avocats

Ce projet est une application interactive développée avec **Dash** et **Plotly** permettant d'analyser les prix et les volumes de ventes d'avocats aux USA.

## Structure du projet
- `app.py` : Point d'entrée de l'application.
- `pages/` : Contenu des différentes pages (Prix, Volume, Infos).
- `datas/` : Jeu de données `avocado.csv`.
- `assets/` : Ressources statiques (images, CSS).
